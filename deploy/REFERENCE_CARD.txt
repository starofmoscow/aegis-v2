┌─ PERFORMANCE TUNING ────────────────────────────────────────────────────────┐
│                                                                              │
│  If CPU high:                                                               │
│    • Check with: make stats                                                │
│    • Scale app horizontally (load balancer needed)                         │
│    • Or increase timeout in nginx.conf                                     │
│                                                                              │
│  If memory high:                                                            │
│    • Check with: make stats                                                │
│    • Reduce NODE_OPTIONS in docker-compose.yml (from 2048 to 1024)        │
│    • Check Redis usage: docker exec aegis-redis redis-cli INFO memory     │
│    • Clear Redis: docker exec aegis-redis redis-cli FLUSHALL              │
│                                                                              │
│  If disk full:                                                              │
│    • Check usage: make disk-usage                                          │
│    • Clean Docker: docker image prune && docker system prune               │
│    • Check log sizes: du -sh /opt/aegis/volumes/*/                        │
│                                                                              │
│  If response slow:                                                          │
│    • Check: nginx cache hits (nginx.conf caching section)                  │
│    • Check: Redis latency (redis-cli --latency)                            │
│    • Check: Upstream API latency (V2Ray logs)                              │
│    • Use: curl -w "@curl-format.txt" https://...                          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
